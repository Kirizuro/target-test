# Test to Target Job Application

## Frontend (Angular 8)

```bash
cd frontend-angular
npm install
ng serve
```

## Frontend (Vuejs)

```bash
cd frontend-vue
yarn
yarn serve
```

## Backend Node (NodeJs + MongoDB)

```bash
cd backend-node
yarn
yarn dev
```

## Backend .Net Core (.Net Core 3.0 + MongoDB)

```bash
cd backend-dotnet
restore
code .
```
